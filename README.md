## Example
![Animation](example/puzzler-animation.gif)
## Building
Run:
```sh
mkdir cmake-build
cd cmake-build
cmake ..
cmake --build .
```
## Note
You can use the [word scrambler](https://github.com/zenon8adams/WordScrambler) program
to generate puzzle files for this program.
